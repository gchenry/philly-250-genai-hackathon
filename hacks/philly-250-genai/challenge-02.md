# Challenge 2: The Visual Blueprint

## Introduction
Turn your script into a visual storyboard.

## Description
Use **Imagen 3 (NanoBanana)** in Vertex AI Studio to generate 16:9 storyboards. 
> **Important:** You must generate a "Contact Sheet" to ensure character consistency.

## Success Criteria
- High-definition storyboards for all 3 scenes.
- A character contact sheet (3x3 or 4x4 grid).
